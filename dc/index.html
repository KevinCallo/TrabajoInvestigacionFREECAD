Proyecto: Scrum y DevOps - FreeCAD

Curso: Ingeniería y Procesos de Software (2026-A)

1. Introducción al Proyecto

Este proyecto implementa un proceso de desarrollo de software que integra el marco de trabajo Scrum con prácticas DevOps. El producto seleccionado es FreeCAD, un modelador 3D paramétrico de código abierto bajo licencia LGPL-2.1.

2. Plan del Proyecto (Scrum + DevOps)

El desarrollo se basa en ciclos iterativos de 15 días. El equipo está compuesto por 5 integrantes con roles rotativos.

Estructura de Sprints

  • Sprint 0: Planificación, análisis del sistema y configuración del entorno.
  • Sprint 1-2: Desarrollo inicial e integración CI/CD.
  • Sprint 3-4: Mejoras, optimización y documentación final.
